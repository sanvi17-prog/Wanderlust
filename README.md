# 🌍 Wanderlust

A full-stack Airbnb-inspired web application where users can explore, create, edit, and review travel listings from around the world.

## 🚀 Live Demo

🔗 https://wanderlust-qzcz.onrender.com/listings

## ✨ Features

- 🔐 User Authentication (Signup, Login & Logout)
- 🏡 Create, Edit & Delete Listings
- 📸 Upload Listing Images
- ⭐ Review & Rating System
- 👤 User Authorization
- 💬 Flash Messages
- 🛡️ Session Management
- 📱 Responsive UI using Bootstrap
- 🗺️ Interactive Maps (Coming Soon)
- 🔍 Search & Filters (Coming Soon)
